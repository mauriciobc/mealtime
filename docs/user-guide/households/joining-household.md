# Entrando em uma Residência

Aprenda como entrar em uma residência existente usando um código de convite.

---

## Usando o Código de Convite

1. Acesse a página **Entrar em um Domicílio** em `/join`
2. Insira o **código de convite** que você recebeu
3. Clique em **"Entrar no Domicílio"**
4. Aceite os termos de uso (se necessário)
5. Você será adicionado à residência

---

## Onde Encontrar o Código?

O código de convite pode ser obtido:
- No email de convite enviado pelo administrador
- Na página de detalhes da residência (para membros existentes)

---

## Problemas Comuns

| Problema | Solução |
|----------|---------|
| Código inválido | Verifique se o código está correto ou solicite um novo |
| Já sou membro | Você já está na residência |
| Residência cheia | Entre em contato com o administrador |
