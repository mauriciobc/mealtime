# Criando Residências

Aprenda a criar e gerenciar residências (lares) no MealTime para compartilhar o cuidado dos gatos com familiares.

---

## O que é uma Residência?

Uma **Residência** (ou Domicílio) é um espaço compartilhado onde:
- Você e sua família podem gerenciar os gatos
- Membros podem ser adicionados via convite
- Cada residência tem seu próprio conjunto de gatos e configurações

---

## Criando uma Nova Residência

![Lista de Residências](/user-guide/assets/screenshots/households.png)

1. Acesse **Domicílios** em `/households`
2. Clique em **"Nova Residência"**
3. Digite um nome para a residência
4. Clique em **"Criar"**

---

## Gerenciando uma Residência

Após criar, você pode:
- **Editar nome e descrição**
- **Convidar membros** via código de convite
- **Ver membros** ativos
- **Sair da residência** (se não for o último admin)

---

## Papéis na Residência

| Função | Permissões |
|--------|------------|
| Admin | Gerenciar tudo, convidar, remover membros |
| Membro | Adicionar gatos, registrar alimentações |

---

## Dicas

- Crie uma residência para cada local onde você cuida de gatos
- Convide familiares que participam dos cuidados
- O código de convite pode ser compartilhado facilmente
