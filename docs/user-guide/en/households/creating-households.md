# Creating Households

Learn how to create and manage households in MealTime to share cat care with family.

---

## What is a Household?

A **Household** is a shared space where:
- You and your family can manage cats
- Members can be added via invitation
- Each household has its own set of cats and settings

---

## Creating a New Household

1. Go to **Households** at `/households`
2. Click **"New Household"**
3. Enter a name for the household
4. Click **"Create"**

---

## Managing a Household

After creating, you can:
- **Edit name and description**
- **Invite members** via invitation code
- **View active members**
- **Leave the household** (if not the last admin)

---

## Roles in Household

| Role | Permissions |
|------|-------------|
| Admin | Manage everything, invite, remove members |
| Member | Add cats, record feedings |

---

## Tips

- Create a household for each place where you care for cats
- Invite family members who participate in care
- The invitation code can be easily shared
