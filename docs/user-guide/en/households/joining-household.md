# Joining a Household

Learn how to join an existing household using an invitation code.

---

## Using the Invitation Code

1. Go to **Join a Household** at `/join`
2. Enter the **invitation code** you received
3. Click **"Join Household"**
4. Accept the terms of use (if necessary)
5. You will be added to the household

---

## Where to Find the Code?

The invitation code can be obtained:
- In the invitation email sent by the administrator
- On the household details page (for existing members)

---

## Common Issues

| Problem | Solution |
|---------|----------|
| Invalid code | Check if the code is correct or request a new one |
| Already a member | You are already in the household |
| Household full | Contact the administrator |
