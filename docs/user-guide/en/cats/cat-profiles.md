# Cat Profiles

Understand all the information available in each cat's profile.

---

## Profile Information

Each cat has a profile with the following information:

| Field | Description |
|-------|-------------|
| Name | Cat's name |
| Photo | Profile picture |
| Current Weight | Last recorded weight |
| Birth Date | Age calculated automatically |
| Meal Interval | Hours between meals |
| Recommended Portion | Standard amount per meal |
| Restrictions | Allergies or foods to avoid |
| Notes | Special notes |

---

## History in Profile

In the cat's profile you will find:

- **Feeding History**: List of all recorded meals
- **Weight Chart**: Weight evolution over time
- **Individual Statistics**: Average portions, feeding frequency

---

## Link to Schedules

Each cat can have multiple feeding schedules. The schedules send notifications at the configured time.
