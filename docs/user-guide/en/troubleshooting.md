# Troubleshooting

Find solutions to the most common MealTime problems.

---

## Common Problems

### Can't Log In

1. Check if your email is correct
2. Confirm your password is correct
3. Try logging in with Google
4. Clear browser cache and try again

### Notifications Not Appearing

1. Check if notifications are enabled in `/settings`
2. Allow notifications in your browser
3. Make sure browser isn't blocking notifications
4. Try logging out and logging in again

### Feeding Not Being Recorded

1. Check if a cat is selected
2. Confirm the portion is set
3. Check your internet connection
4. Try refreshing the page

### Cat Image Not Loading

1. Check if the image URL is correct
2. Try uploading a new photo
3. Format must be JPG or PNG
4. Maximum size is 5MB

### Household Issues

1. Check if you have admin permission
2. Confirm the invite code is correct
3. Try leaving and joining the household again

---

## Weight Tracking Issues

### Weight Not Saving

1. Make sure a cat is selected
2. Check the weight value is valid (positive number)
3. Verify your internet connection
4. Try refreshing the page

### Weight Goals Not Working

1. Verify the goal has a target weight set
2. Check the deadline date is in the future
3. Make sure the cat is assigned to the goal
4. Try recreating the goal

---

## Schedule Issues

### Schedules Not Triggering

1. Check if the schedule is active
2. Verify the scheduled time is correct
3. Make sure notifications are enabled
4. Check if the schedule has a cat assigned

### Not Receiving Reminders

1. Confirm push notifications are enabled in browser settings
2. Check if notifications are enabled in `/settings`
3. Make sure the schedule time hasn't passed
4. Verify the schedule is active

---

## Account Issues

### Can't Create Account

1. Use a valid email address
2. Password must be at least 6 characters
3. Try using a different browser
4. Disable browser extensions temporarily

### Can't Update Profile

1. Check all required fields are filled
2. Verify the username is available
3. Try updating one field at a time
4. Clear browser cache and try again
