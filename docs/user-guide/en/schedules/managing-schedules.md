# Managing Schedules

Learn how to create and manage feeding schedules for your cats.

---

## Creating a Schedule

1. Go to **Schedule** at `/schedules`
2. Click **"New Schedule"**
3. Configure the schedule:

| Field | Description |
|-------|-------------|
| Cat | Select the cat |
| Time | Feeding time |
| Days of Week | Which days to activate |
| Repeat | Repeat interval |
| Active | Turn schedule on/off |

4. Click **"Save"**

---

## Editing Schedules

1. Go to the schedules list
2. Click on the schedule you want to edit
3. Modify the settings
4. Save the changes

---

## Activating/Deactivating

You can quickly activate or deactivate a schedule:
- Click the toggle button on the schedule card
- Inactive schedules don't send notifications

---

## Notifications

When a schedule is active:
- You will receive a **push notification** at the scheduled time
- The notification reminds you to feed the cat
- You can record the feeding directly from the notification

---

## Tips

- Set up multiple schedules for different times
- Use schedules for routine (morning, afternoon, evening)
- Enable push notifications so you don't forget
