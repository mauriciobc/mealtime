# Gerenciando Agendamentos

Aprenda a criar e gerenciar agendamentos de alimentação para seus gatos.

---

## Criando um Agendamento

![Página de Agendamentos](/user-guide/assets/screenshots/schedules.png)

1. Acesse **Agenda** em `/schedules`
2. Clique em **"Novo Agendamento"**
3. Configure o agendamento:

| Campo | Descrição |
|-------|-----------|
| Gato | Selecione o gato |
| Horário | Hora da alimentação |
| Dias da Semana | Quais dias ativar |
| Repetir | Intervalo de repetição |
| Ativo | Ligar/desligar o agendamento |

4. Clique em **"Salvar"**

---

## Editando Agendamentos

1. Vá para a lista de agendamentos
2. Clique no agendamento que deseja editar
3. Modifique as configurações
4. Salve as alterações

---

## Ativando e Desativando

Você pode rapidamente ativar ou desativar um agendamento:
- Clique no botão de alternar (toggle) no card do agendamento
- Agendamentos inativos não enviam notificações

---

## Notificações

Quando um agendamento está ativo:
- Você receberá uma **notificação push** no horário agendado
- A notificação lembra você de alimentar o gato
- Você pode registrar a alimentação diretamente da notificação

---

## Dicas

- Configure múltiplos agendamentos para diferentes horários
- Use agendamentos para rotina (manhã, tarde, noite)
- Ative as notificações push para não esquecer
