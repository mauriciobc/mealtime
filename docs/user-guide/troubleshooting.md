# Solução de Problemas

Encontre soluções para os problemas mais comuns do MealTime.

---

## Problemas Comuns

#### Não consigo fazer login
1. Verifique se seu email está correto
2. Confirme que a senha está correta
3. Tente usar o login com Google
4. Limpe o cache do navegador e tente novamente

#### Notificações não aparecem
1. Verifique se as notificações estão ativadas em `/settings`
2. Permita notificações no seu navegador
3. Verifique se o navegador não está bloqueando notificações
4. Tente fazer logout e login novamente

#### Alimentação não está sendo registrada
1. Verifique se o gato está selecionado
2. Confirme que a porção está definida
3. Verifique a conexão com a internet
4. Tente atualizar a página

#### Imagem do gato não carrega
1. Verifique se a URL da imagem está correta
2. Tente enviar uma nova foto
3. O formato deve ser JPG ou PNG
4. O tamanho máximo é 5MB

#### Problemas com residência
1. Verifique se você tem permissão de administrador
2. Confirme que o código de convite está correto
3. Tente sair e entrar na residência novamente
