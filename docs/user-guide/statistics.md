# Estatísticas

Visualize gráficos e dados detalhados sobre a alimentação dos seus gatos.

---

## Acessando Estatísticas

![Página de Estatísticas](/user-guide/assets/screenshots/statistics.png)

1. Vá para **Estatísticas** em `/statistics`
2. Visualize os gráficos e dados
3. Use filtros por período e gato

---

## Dados Disponíveis

Na página de estatísticas você encontra:

- **Total de Alimentações**: Número de refeições registradas
- **Porção Média**: Quantidade média por refeição
- **Gráfico de Tendências**: Evolução ao longo do tempo
- **Por Gato**: Estatísticas individuais por gato

---

## Interpretando os Dados

| Métrica | Significado |
|---------|-------------|
| Total de Alimentações | Refeições registradas no período |
| Porção Média | Quantidade média por refeição |
| Tendência | Se o gato está comendo mais ou menos |

---

## Sem Dados

Se a página mostrar "Sem Dados":
- É necessário registrar mais alimentações
- Os dados aparecem após alguns registros
