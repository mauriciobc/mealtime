# Perfis dos Gatos

Entenda todas as informações disponíveis no perfil de cada gato.

---

## Informações do Perfil

Cada gato possui um perfil com as seguintes informações:

| Campo | Descrição |
|-------|-----------|
| Nome | Nome do gato |
| Foto | Imagem de perfil |
| Peso Atual | Último peso registrado |
| Data de Nascimento | Idade calculada automaticamente |
| Intervalo de Refeições | Horas entre cada refeição |
| Porção Recomendada | Quantidade padrão por refeição |
| Restrições | Alergias ou alimentos a evitar |
| Observações | Notas especiais |

---

## Histórico no Perfil

No perfil do gato você encontra:

- **Histórico de Alimentações**: Lista de todas as refeições registradas
- **Gráfico de Peso**: Evolução do peso ao longo do tempo
- **Estatísticas Individuais**: Média de porções, frequência de alimentações

---

## Vincular a Agendamentos

Cada gato pode ter múltiplos agendamentos de alimentação. Os agendamentos enviam notificações no horário configurado.
